# Interactive-dashboard

## Overview
A responsive dashboard built with React + TypeScript featuring dynamic data visualization, authentication, and theme toggling.

## Features
- User authentication (JWT-based)
- Interactive charts (Chart.js)
- Dark/light mode support
- Responsive design with TailwindCSS

## Tech Stack
- React + TypeScript
- Chart.js
- TailwindCSS
- Mock API (JSON Server)
